# 💧 AquaTrack - Smart Water Tracker

A React-based web application to track daily water intake, set hydration goals, and view analytics with beautiful charts.

## 📌 What This Website Does

Users can track how much water they drink daily. Set a target (like 3 liters per day). Add water with one click. See progress in real-time. View weekly reports and charts to analyze hydration patterns.

## 🛠️ Tech Stack Used

- React 18
- Vite
- React Router DOM
- CSS3
- LocalStorage

## ✨ Features

- Add water: 200ml, 500ml, 1000ml, or custom amount
- Set daily target: 500ml to 5000ml
- Progress bar shows percentage
- History of today's intake with delete option
- Celebration alert when goal reached
- Weekly bar chart
- Monthly trend view
- Data automatically saved in browser
